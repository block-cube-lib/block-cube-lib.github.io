Git Author

git authorの日本語ドキュメントです。
リポジトリ

Overview

  • コマンド一発でuser.nameとuser.emailを取得または設定できます
  • 過去のコミットのAuthorを書き換えることができます

Description

git authorまたはgit author getでuser.nameとuser.emailを取得できます。

git set foo foo@abc.comでuser.nameとuser.emailを設定できます。

git replace foo foo@abc.com bar bar@xyz.comで 過去のコミットのauthorとcommiterのfoo <foo@abc.com>をbar <bar@xyz.com>に置き換えます。
置き換え後の名前とメールアドレス(bar bar@xyz.com)は省略可能で、省略した場合はgit author getで取得できるauthorに置き換えます。

Installation

Cargoが使える必要があります。
cargo install git-author でインストールできます。
