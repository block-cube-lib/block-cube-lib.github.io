ブログをGitHub Actionsでデプロイするようにしました

2021/06/10 20:48
GitHub Actions zola

今までのこのブログは

  1. ローカルでMarkdownを編集
  2. Markdownの変更をプライベートリポジトリにpush
  3. Zolaでhtmlに変換
  4. ビルドしたものをGitHub Pagesで公開しているリポジトリにpush

という流れでデプロイしており2,4でそれぞれ別リポジトリにpushする必要がありました。
これが結構面倒だったので、GitHub Actionsを使用して3,4を自動化しました。

やったことはZennにZolaで作ったサイトをGitHub ActionsでGitHub PagesにDeployするという記事を書いたのでそちらを見てください。
