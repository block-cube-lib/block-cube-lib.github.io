esp-idf-svc(Rust + ESP32)のプロジェクトのRustのバージョンを上げる

2026/02/05 23:02 公開
ESP32 Rust esp-idf-svc

プロジェクトのrust-toolchain.tomlで以下のようにchannelが指定されているものが対象です。

[toolchain]
channel = "esp"

以下のコマンドでRustのバージョンを更新できます。

espup update

stableやnightlyのバージョンを更新する場合は下記のコマンドで良いですが、channelがespのRustのバージョンは上がりません。

rustup update
